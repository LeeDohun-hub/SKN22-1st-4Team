## 중고차 리콜 사유 분석 시스템

Streamlit 기반의 중고차 리콜 정보 분석 웹 애플리케이션입니다.

### 주요 기능

1. **상단 요약바**
   - 검색 기능 (차명/제조사)
   - 필터링 (제조사, 차종, 연도, 리콜 사유)
   - 요약 통계 (총 리콜 건수, 평균 시정률, 시정률 50% 이하 건수 등)

2. **중단 영역**
   - 카드형/표형 리콜 목록 표시
   - 차종별 리콜 내역 한눈에 보기
   - 시정률에 따른 색상 구분

3. **하단 상세 정보**
   - 선택한 리콜의 상세 정보 표시
   - 리콜 번호, 발표일자, 상세 사유
   - 대상 차량 정보, 조치 방법, 연락처

### 설치 및 실행

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. Streamlit 앱 실행:
```bash
streamlit run app.py
```

3. 브라우저에서 자동으로 열리는 페이지에서 사용하세요.

### 기술 스택

- Python 3.8+
- Streamlit
- Pandas

### 데이터 구조

- 제조사 정보 (현대, 기아, BMW, 볼보, 테슬라 등)
- 모델 정보 (각 제조사별 차종)
- 리콜 데이터 (리콜 사유, 시정률, 조치 상태 등)
